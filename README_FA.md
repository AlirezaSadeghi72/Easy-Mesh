# ایزی مش چیست؟

یک اسکریپت ساده برای راه اندازی شبکه مش بر اساس هسته easytier

## نحوه ی نصب


اسکریپت زیر را در سرور خود (دبیان یا اوبونتو) اجرا کنید:

```
bash <(curl -Ls --ipv4 https://github.com/AlirezaSadeghi72/easy-mesh/raw/main/easymesh.sh)
```
